# PDF to excel converter

This repository contains Python scripts and Jupyter notebooks for PDF to Excel conversion and file system comparison tasks.

## Files

- `Amazon payment reports pdf to excel.py` - Convert Amazon payment report PDFs to Excel format
- `FS_comparison.py` - Financial Statement comparison utility (this is not working hence, added to ignore)
- `pdf2excel_1 file.py` - Single file PDF to Excel conversion
- `pdf2excel_multiple_files_auto_country.py` - Batch PDF to Excel conversion with automatic country detection
- `test.ipynb` - Jupyter notebook for testing

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the Python scripts directly or open the Jupyter notebook in your preferred environment.
